Connexion Strava

Pour afficher vos sorties, vous avez besoin d'un Access Token Strava. Suivez les 2 étapes ci-dessous :

① Autoriser l'accès

Cliquez pour autoriser la connexion à votre compte Strava :

AUTORISER SUR STRAVA →

Après autorisation, copiez le code=XXXXX depuis l'URL de la page d'erreur localhost.

② Obtenir le token (curl)

Dans votre terminal, remplacez VOTRE_CODE :

curl -X POST https://www.strava.com/oauth/token \ -d client_id=43268 \ -d client_secret=16c230638da8ab77494cf8ee8c3597b07e8057a0 \ -d code=VOTRE_CODE \ -d grant_type=authorization_code

La réponse JSON contient "access_token":"..." — copiez cette valeur.

🔒 Votre token n'est jamais envoyé ailleurs que vers l'API Strava officielle.
