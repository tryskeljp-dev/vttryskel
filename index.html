Connexion Strava

Suivez ces 3 étapes pour charger vos sorties vélo 2025.

① Autoriser l'accès Strava

Cliquez sur le bouton, autorisez l'accès, puis copiez le code dans l'URL (après code=)

AUTORISER SUR STRAVA →
② Obtenir le token (Windows : ouvrez cmd avec Windows+R puis tapez cmd)

Collez cette commande dans le terminal en remplaçant VOTRE_CODE :

curl -X POST https://www.strava.com/oauth/token -d "client_id=43268&client_secret=16c230638da8ab77494cf8ee8c3597b07e8057a0&code=VOTRE_CODE&grant_type=authorization_code"

Copiez la valeur de "access_token" dans la réponse JSON.

③ Collez votre access token

🔒 Votre token n'est jamais envoyé ailleurs que vers l'API Strava officielle.
